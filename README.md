# AniyomiJPDB

### A fork of Aniyomi, a full-featured player and reader with JPDB integration, based on ~~Tachiyomi~~ Mihon.
Discover and watch anime, cartoons, series, and more – easier than ever on your Android device.

## JPDB Setup

To use the JPDB vocabulary lookup feature:

1. **Get your JPDB API Key**:
   - Visit [jpdb.io](https://jpdb.io)
   - Create or log in to your account
   - Navigate to your account settings
   - Copy your API key

2. **Configure in Aniyomi**:
   - Open the app and go to More (⋯) menu
   - Select "JPDB"
   - Tap "Put your JPDB API key"
   - Paste your API key and save

3. **Use it while watching**:
   - While watching anime with subtitles, tap the "Parse" button in the player controls
   - Vocabulary from the current subtitle will be parsed and displayed
   - Swipe up on the overlay to dismiss it, or wait for auto-dismiss

For more information, visit [JPDB API Documentation](https://jpdb.stoplight.io/docs/jpdb/609bb98ccd378-parse-text)

## Features

<div align="left">

* Local reading and watching of content.
* A configurable reader with multiple viewers, reading directions and other settings.
* A configurable player built on mpv-android with multiple options and settings.
* **JPDB Integration**: Parse Japanese vocabulary from anime subtitles with JPDB API support.
  * Real-time Japanese text parsing with the press of a button
  * Vocabulary lookup with readings, meanings, and definitions
* Tracker support: [MyAnimeList](https://myanimelist.net/), [AniList](https://anilist.co/), [Kitsu](https://kitsu.app/), [MangaUpdates](https://mangaupdates.com), [Shikimori](https://shikimori.one), [Simkl](https://simkl.com/), and [Bangumi](https://bgm.tv/) support.
* Categories to organize your library.
* Light and dark themes.
* Schedule updating your library for new chapters/episodes.
* Create backups locally to read/watch offline or to your desired cloud service.
* Plus much more...
